# NSString+RemoveEmoji

A category on NSString that extends for removing Emoji.

## Usage

```objectivec
BOOL isIncludingEmoji = [@"✨Party✨ ✨Hard✨" isIncludingEmoji];
NSString* removedEmoji = [@"✨Party✨ ✨Hard✨" removedEmojiString];
```
